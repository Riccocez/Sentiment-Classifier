Project focused on developing simple sentiment classifiers of US Airline tweets by using ML techniques.

The dataset consists of Tweets of six US Airlines from February 2015. Data contain 3 main classes:

1) Positive
2) Negative
3) Neutral

followed by categorising negative reasons.

Fields in the database table are:

	-tweet_id
	-airline_sentiment
	-airline_sentiment_confidence
	-negativereason
	-negativereason_confidence
	-airline
	-airline_sentiment_gold
	-name
	-negativereason_gold
	-retweet_count
	-text
	-tweet_coord
	-tweet_created
	-tweet_location
	-user_timezone

Dataset can be found at: 

https://www.kaggle.com/crowdflower/twitter-airline-sentiment
